    How to Python make OpenStack better (and vice-versa)

    Samuel Medeiros

    O OpenStack é um sistema operacional em nuvem que controla grandes polos de recursos de computação, armazenamento e rede em um datacenter, tudo gerenciado através de um painel que dá aos administradores controle ao mesmo tempo que capacita seus usuários para fornecer recursos através de uma interface web. Ele agora consiste em mais de 4,5 milhões de linhas de código, 85% das quais é Python.
    Por que o OpenStack originalmente escolheu o Python como seu idioma principal? O que o Python traz para o OpenStack? Por outro lado, como o OpenStack ajuda bibliotecas Python? Como o sistema de integração contínua baseado em nuvem que o OpenStack usa expõe bugs em bibliotecas Python? Esta palestra dará respostas a todas essas perguntas, enquanto destacando como Python é importante para o OpenStack, e vice-versa.

    Um jeito Python de ser

    Matheus Gaudencio

    Python é uma linguagem de programação amigável e direta. Quem aprende Python costuma ter domínio sobre as construções básicas da linguagem mas acaba não conhecendo um outro lado dessa linguagem. Existem construções, bibliotecas, utilitários e até comandos básicos que, se usados corretamente, tornam o código Python mais legível, direto e simples. Nesta apresentação discutiremos sobre o jeito Python de escrever código, alguns recursos como list comprehensions, generators e iterators, bibliotecas de manipulação de funções, utilitários para manipular string de forma direta, e outras funcionalidades para se tornar um Pythonista de fato.
